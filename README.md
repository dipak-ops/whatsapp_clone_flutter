# WhatsApp Clone 📱

A Flutter-based WhatsApp Clone that recreates the core functionality and user experience of WhatsApp. This project was developed to practice Flutter development, UI design, navigation, custom widgets, and package integration.

## 📖 Description

WhatsApp Clone is a cross-platform mobile application built using Flutter. The application includes major WhatsApp features such as Chats, Status, Calls, Contacts, and Camera screens. It provides a responsive and modern user interface while demonstrating Flutter best practices and clean project structure.

The project uses mock data to simulate real-world messaging scenarios, making it suitable for learning Flutter development and understanding mobile application architecture.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/whatsapp-clone.git
```

### Navigate to Project

```bash
cd whatsapp-clone
```

### Install Dependencies

```bash
flutter pub get
```

### Run App

```bash
flutter run
```
